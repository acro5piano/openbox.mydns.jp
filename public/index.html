Openbox

An open-source, lightweight, highly-customizable window manager for X

Why Openbox? See screenshots

Install with one command

You can install Openbox with one command. Paste at a Terminal prompt:

Ubuntu 16.04~

sudo apt install openbox

Debian and Ubuntu ~15.10

sudo apt-get install openbox

Fedora

sudo yum install openbox

Extremely light

Openbox is lightweight. It uses memory less than 100MB. Instead of abusing 3D effect, GUI configuration, Clear-look theme, You can use your PC more productively.
